# Weather-App
A sleek and responsive Weather Application that provides real-time weather data using the OpenWeatherMap API.

Tech Stack
HTML5: Semantic structure.

CSS3: Custom styling with Glassmorphism effects and Responsive Design.

JavaScript (ES6+): Asynchronous API fetching and DOM manipulation.

 API: OpenWeatherMap API for live data.

# Features
Real-time Search: Get weather updates for any city in the world.

Dynamic Data: Displays Temperature, Humidity, and Wind Speed.

Interactive UI: Glassmorphism design with hover effects and "Enter" key support.

Error Handling: Built-in alerts for invalid city names.
